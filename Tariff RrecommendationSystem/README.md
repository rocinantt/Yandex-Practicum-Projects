# Классификация клиентов телеком компании

## Описание проекта
Цель проекта - построить модель для классификации клиентов телеком компании и предложить им один из новых тарифов на основе анализа их поведения. Для этого используются данные о клиентах и их активности.

## Сфера деятельности
Телеком

## Цели проекта
Определить оптимальный тариф для клиентов телеком компании на основе их поведения.

## Стек и библиотеки
Matplotlib, Pandas, Python, Scikit-learn

## План проекта
1. **Загрузка и подготовка данных**
2. **Обучение и исследование моделей**
3. **Тестирование модели**

## Финальный вывод
В результате исследования были обучены и проанализированы три модели для классификации клиентов телеком компании. Лучшей моделью оказался случайный лес с точностью (accuracy) 82.4%.
